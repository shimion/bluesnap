== Description ==

Bluesnap for WordPress allows seamless Bluesnap Payment Gateway IPN integration for WordPress blogs.

<strong>Main Features:</strong>
<ul>
	<li>Receive IPN (instant payment notification) directly in your WordPress blog</li>
	<li>Admin dashboard with charts and reports</li>
	<li>Automatically register buyers as users on the blog</li>
	<li>Send the buyers automatic email</li>
</ul>

Bluesnap for WordPress relies on Bluesnap IPN (instant payment notification) to provide real time reports about your sales. It can also automatically register new buyers as users on your blog which allows you to for example use membership plugins.

For more information visit <a href="http://www.prelovac.com/vladimir/wordpress-plugins/plimus-for-wordpress">Bluesnap for WordPress plugin page</a>.

== Installation ==

1. Upload the whole plugin folder to your /wp-content/plugins/ folder.
2. Go to the plugins page and activate the pluginl
3. Visit plugin options (Bluesnap IPN/Options) and copy the URL to the plimusipn.php file and paste it in your Bluesnap General Settings page.
